# Markdown Air

Lightweight local markdown editor for Windows built with Tauri + Vanilla TypeScript.

## Features

- Open / Save / Save As for local `.md` files
- Plain text markdown editing
- Unsaved changes prompt when closing
- Markdown preview toggle (raw HTML disabled)
- Always on top toggle with persisted preference

## Development

Prerequisites:

- Node.js
- Rust toolchain
- Tauri prerequisites for Windows

Commands:

```bash
npm install
npm run tauri dev
```

Build:

```bash
npm run build
npm run tauri build
```
